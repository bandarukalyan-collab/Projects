# Generalized Video Transcript Analysis Prompt

## Purpose
This prompt is designed to analyze video transcripts of any type (interviews, tutorials, meetings, presentations, etc.) and generate a comprehensive, structured summary adapted to the video format.

## Instructions

Analyze this video transcript and generate a comprehensive summary. Adapt the output format based on the video type:

### If the transcript contains Q&A (interviews, quiz sessions):
- Extract all questions and answers
- Format as numbered Q1, Q2, Q3...
- Provide detailed bullet-point answers

### If the transcript is educational/tutorial:
- Extract step-by-step instructions
- List key concepts explained
- Include code examples or technical details

### If the transcript is a meeting/discussion:
- Extract action items and tasks
- List decisions made
- Identify key participants and their roles

### If the transcript is a presentation:
- Summarize main points
- List key takeaways
- Extract notable quotes

### For all video types, include:
1. **Overview** - What is this video about? (2-3 sentences)
2. **Main Topics** - Key subjects covered
3. **Key Points** - Important information or insights
4. **Technical Details** - Tools, technologies, methods mentioned
5. **Notable Quotes** - Memorable statements

Format the output clearly with headings for easy reading.
